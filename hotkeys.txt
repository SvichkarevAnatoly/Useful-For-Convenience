-----Vimperator(возможно для всех браузеров работает)-----
u - восстановление закрытой вкладки
b - открывает поиск по открытым вкладкам
F6 | Ctrl+l - выделяет для редактирования строку адреса
Ctrl+o - вернуться на предыдущую страницу во вкладке
Ctrl+6 - вернуться на прошлую открытую вкладку
gg - перейти в начало страницы
[процент]gg - переход к месту на странице по проценту
Ctrl+d - промотать вниз полстраницы
Ctrl+u - промотать вверх полстраницы

-----Idea-----
Ctrl+Shift+a - поиск по командам
Ctrl+Shift+F12 - максимизировать область редактирования
Shift+Enter - перейти на новую строку с нужным отступом из любого места текущей строки
Ctrl+j - показать список темплейтов
Ctrl+y - удалить строку
Ctrl+Shift+Enter - автокомплит с переходом на новую строку
Alt+Enter - диалог возможного исправления ошибок
Ctrl+Alt+v - автоматическое заведение переменной
Ctrl+w - выделение смысловой области
Shift+Esc - скрыть активную область
Alt+цифра - открыть пронумерованную панель(повторное нажатие скрывает)
Ctrl + наведение мышью на метод - открыть декларацию
Shift+F10 - запуск программы
Ctrl+/ - закоментировать однострочным комментарием
Ctrl+Shift+/ - закоментировать многострочным комментарием
Shift+F6 - переименовать сущность везде
Shift+Shift - искать по всему
F2 - переход к следующей ошибке или варнингу

-----Eclipse-----
Ctrl+d - удалить строку
Ctrl+i - коррекция отступов
Ctrl+j - дополняющий поиск
Ctrl+k - находит следующее вхождение без открытие окна поиска
Shift+Enter - переходит на новую строчку без разрыва текущей(если посередине строки стоишь)

-----Ubuntu-----
Alt+F10 - Вызывает меню у текущего окна
Ctrl+Super+Down - Свернуть(уменьшить) рабочее окно

-----Терминал-----
Ctrl+a - перемещает курсор в начало строки
Ctrl+e - перемещает курсор в конец строки
Ctrl+l - очистить терминал
Shift+Ctrl+t - открыть новую вкладку
Shift+Ctrl+w - закрыть текущую вкладку
Ctrl+PgUp | Ctrl+PgDn - переключаться по вкладкам

-----Терминал(полезные команды)-----
---Программы---
eog имя_картинки - просмотр картинки

---Git---
git checkout HEAD имя_файла - удаляет изменения указанного файла
git diff HEAD^ - показывает изменения внесённые последним коммитом

-----Nautilus-----
Ctrl+Up - подняться наверх по директориям
Ctrl+Down - вернуться вперёд по истории директорий
