" ========== Tcomment ================================
gcc - комментирование текущей строки
gc - раскомментирование текущей строки
gcip - закомментировать текущий блок строк, повторно для раскомментаривания

" ========== Neoterm =================================
:Topen - закрывает существующий терминал
:Tmap - позволяет создавать команду которая будет отправляться терминалу при нажатие сочетания
,tt - запуск заданной через Tmap команды

" ========== Проверка синтаксиса (не плагин) =========
]s - переход к следующему слову с ошибкой
[s - переход к предыдущему слову с ошибкой
z= - выбор исправления ошибки из вариантов из списка
F2 - вкл/выкл проверку синтаксиса
zg - добавить слово в словарь
zG - добавить слово в словарь только для текущего сеанса
zug - удалить слово из словаря
