break номер - поставить брейкпойнт на указанную строчку
run - запустить программу
p переменная - показать значение переменной
quit - выход из дебагера
l - показывает какой-то участок кода
l число:число - показывает диапазон строк в текущем файле
c - продолжить исполнение
s - следующая инструкция, если функция, то заход внутрь
n - следующая инструкция без захода внутрь
Enter - повторить прошлую команду
help команда - вывести справку по команде
help - вывести общую справку
bt - вывести стек вызовов
bt full - вывести стек со значениями текущих переменных

gdb -tui - запуск дебагера с терминальным графическим интерфейсом
